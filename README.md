# Advent of Code 2024
Solve daily coding challenges in Rust! Each challenge is implemented in its own file, and you can easily add your solutions and run them.

[Advent of Code 2024](https://adventofcode.com/)

## Directory Structure
```
src/
├── days/
│   ├── day1.rs
│   ├── day2.rs
│   ├── ...
│   ├── day25.rs
│   └── mod.rs
├── main.rs
```
## Getting Started
1. Clone the Repository:

```
git clone https://github.com/brentrill/Advent2024.git
cd Advent2024
```
2. Create Your Branch:

```
git checkout -b <your-branch-name>
```
3. Run a Challenge: Use the following command to run a specific day's challenge:

```
cargo run -- <day>
```
  Example:

```
cargo run -- 3
```
